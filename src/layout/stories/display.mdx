# Display

Utilities for controlling the display box type of an element.

| Utility       | Properties               |
| ------------- | ------------------------ |
| `block`       | `display: block;`        |
| `inlineBlock` | `display: inline-block;` |
| `inline`      | `display: inline;`       |
| `flex()`      | `display: flex;`         |
| `inline-flex` | `display: inline-flex;`  |
| `table`       | `display: table;`        |
| `tableRow`    | `display: table-row;`    |
| `tableCell`   | `display: table-cell;`   |
| `hidden`      | `display: none;`         |

## Hidden

Use `hidden` to set an element to `display: none` and remove it from the
page layout (compare with `invisible` from the [visibility](#) documentation).

## Responsive

To control the display property of an element at a specific breakpoint,
add a `{screen}()` wrapper to any existing utility. For example, use
`md(inlineFlex)` to apply the `inlineFlex` utility at only medium screen
sizes and above.
